# Support

Use the issue template that matches what happened. It keeps reports short and makes fixes faster.

## Best Places To Start

| Need | Where to go |
| --- | --- |
| Something broke | Bug report |
| A game is not detected | Game detection report |
| Installer, startup, or update problem | Installer and update report |
| A new setting or workflow would help | Feature request |
| Shared profile suggestion | Community profile report |

## Helpful Details

Include the OMNAFK version, Windows version, game or app name, executable name, and a screenshot when it helps.

For installer problems, include whether the setup window opened, whether Windows showed a prompt, and whether OMNAFK appears in the Start menu or tray.

For game detection problems, include the game executable name if you can see it in Sightline.

## Logs

If a log file is available from Settings, attach only the part that matches the issue. Avoid sharing private paths or account names when they are not needed.
