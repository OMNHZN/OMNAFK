# Code Of Conduct

OMNAFK should be a useful, calm project space for people reporting issues, suggesting improvements, and sharing game-profile details.

## Expected Behavior

- Be respectful and direct.
- Assume reports are made in good faith.
- Keep issue comments focused on reproducing, fixing, or clarifying the problem.
- Avoid posting private information, tokens, account names, or unnecessary system details.

## Not Accepted

- Harassment, threats, or personal attacks.
- Spam or repeated off-topic comments.
- Posting exploit details publicly when a private security report is more appropriate.
- Sharing another person's private information.

## Enforcement

Project maintainers may edit, hide, lock, or remove comments that make the project harder to use or less safe for others. Serious issues can be reported privately through GitHub Security Advisories when they involve installer, update, privilege, or execution-safety concerns.
