# OMNAFK v0.1.18

This release makes OMNAFK smarter about when a game is actually in session, at a menu, or safe to keep alive.

## What's New

- Added local presence sensing for menu/lobby vs in-session states.
- Added Sightline presence details for marked game targets.
- Added community presence rules for GTA V and Minecraft variants.

## Improved

- Hold while I'm playing now only holds the focused game, so input in another app does not pause background keepalives.
- Background keepalive ticks wait for a quiet input moment before a focus flick.
- Tray eyes stay steadier with less repaint flicker.
- Memory-based presence reads are expert opt-in.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
