## v0.1.3

This release adds adaptive keepalive actions. OMNAFK can learn a small per-game movement profile from safe keys, then use that profile once it has enough samples.

### Added

- Adaptive actions setting for learned per-game keepalives.
- Per-target learned action summary in Sightline details.
- Reset control for a target's learned action profile.

### Notes

- The stable update channel remains GitHub Releases.
- Local settings and learned profiles stay on the machine.
