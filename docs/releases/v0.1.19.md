# OMNAFK v0.1.19

This release tightens the flyout layout, makes target controls easier to manage, and gives holding a quieter tray state.

## Changes

- Refined the top bar so navigation, version, and pin controls share one cleaner row.
- Moved Sightline actions into a compact menu with confirmation for broad changes.
- Added a Smart presence master control with Tune options for the individual detection layers.
- Collapsed lower-frequency Settings groups to keep daily controls easier to scan.
- Changed holding in the tray to a steady half-awake icon.
- Sends engine notices to native Windows notifications so they remain visible in Action Center.

## Install

Download `OMNAFK-Setup.exe` from this release and run it. Existing settings are kept.
