# OMNAFK v0.1.15

This release tightens false-positive control and smooths a few daily settings.

## What's New

- Added **Always ignore exes** for windows that should never be treated as games.
- Added more hold-window choices, from 15 seconds up to 10 minutes.
- Moved Hold time next to Hold while I'm playing so the timing is easier to find.

## Improved

- Expanded built-in filtering for common chat, media, creative, editor, and terminal apps.
- Sightline explanations now mention always-ignore exe rules when they decide a target.
- Custom interval input now stays visible while you are entering a custom value.
- Pending update install now starts setup before stopping the keepalive engine, so a failed installer launch does not leave the app half-stopped.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
