# OMNAFK v0.1.17

This release makes updates reach long-running tray sessions, not only fresh launches.

## What's New

- OMNAFK now re-checks stable GitHub Releases periodically while it stays open in the tray.
- Automatic update mode still checks and installs stable updates even if launch notifications are turned off.

## Improved

- Existing users on the old default update prompt are moved to automatic updates.
- Manual-only and quieter prompt choices are preserved.
- Update install is guarded so a manual click and an automatic check cannot launch duplicate installers.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
