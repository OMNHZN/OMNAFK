# OMNAFK v0.1.10

This release makes OMNAFK faster to understand on first launch and more automatic for common games.

## What's New

- Sightline now opens first so targets and detection status are immediately visible.
- Common games get built-in keepalive hints, including GTA V, Minecraft, Fortnite, Valorant, CS2, and other platform titles.
- Adaptive actions can become ready after 20 samples instead of 50.
- OMNAFK can prompt to restart elevated when a marked game is running as administrator.
- Settings includes a toggle for automatic elevation prompts.

## Improved

- Keepalive controls are grouped around timing, input, behavior, and advanced options.
- Stats hides deeper logs until they are needed.
- Target details show clearer summaries before diagnostic details.
- The community manifest includes more game hints and updated delivery settings for platform titles.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
