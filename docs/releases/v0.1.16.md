# OMNAFK v0.1.16

This release makes first setup clearer, keeps important targets easier to reach, and simplifies notifications.

## What's New

- Added starred Sightline targets. Starred games float above the rest of the target list.
- Added a first-run walkthrough for Sightline, keepalive timing, safety controls, and basic terms.
- Added a replay walkthrough action in About.
- Stable updates are now checked and installed automatically on launch by default when OMNAFK is idle.

## Improved

- Closed-flyout notices now use native Windows notifications.
- In-flyout notices still keep their action buttons when the flyout is open.
- Removed the old tray-adjacent toast window and cleaned up related IPC/docs.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
