## v0.1.9

This release changes keepalive delivery so marked games use real keyboard and mouse input by default.

### Improved

- Keepalives now use direct input when the target is already foreground.
- Background targets use a brief focus flick before sending input, then return control.
- The old background-message path is still available as Background-only mode for games that support it.
- Existing installs migrate away from the old background-only default once, while preserving the setting after a user turns it back on.
- Recovery messages now describe the camera nudge and mouse wiggle fallback path more clearly.
