## v0.1.6

This release adds safer recovery options when a target rejects keepalives, plus faster setup controls for common play styles.

### Added

- Quick presets for common keepalive setups.
- Keepalive health warnings after repeated send failures.
- Auto-fallback for trying focus-flick mode and camera nudge during the current session.
- Per-target adaptive controls and success-rate details.
- Immediate monitor move control from Sightline details.
- GPU activity as an extra detection signal.
- Always-mark process names for games that need a manual detection hint.
- Headless tray mode.
- Restart as administrator from Settings for elevated target apps.

### Improved

- Adaptive learning can use configurable sample thresholds, short key sequences, and successful action history.
- Sightline shows more detail about failing keepalives, alternate windows, GPU activity, and admin mismatch.
- The tray menu includes monitor placement status when placement is active.
- The website FAQ now covers admin mode, fallback behavior, and always-marked games.
