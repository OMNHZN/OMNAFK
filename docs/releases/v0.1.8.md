## v0.1.8

This release adds opt-in community game profiles while keeping local control as the default.

### Added

- Community game profiles can provide shared detection hints and proven keepalive settings.
- Sightline can show when a shared profile is available or applied.
- Community behavior is documented in `docs/COMMUNITY.md`.

### Improved

- Community client IDs are created only after the feature is enabled.
- Auto-applied community profiles are saved immediately.
- Shared profile data cannot turn off quiet background sending or change global monitor placement preferences.
