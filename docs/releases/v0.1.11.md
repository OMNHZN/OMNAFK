# OMNAFK v0.1.11

This release tightens keepalive timing and makes the active status line easier to read.

## What's New

- OMNAFK now tracks its own successful SendInput tick so play-aware holding does not treat that injected input as player activity.
- Adaptive learning now ignores OMNAFK's own keepalive tick and keeps watching for real user input.
- The bottom status bar separates the current state from the next-tick timer.

## Improved

- Key, mouse, wheel, and right-click actions are sent with paced down/up events instead of one compact input batch.
- Focus-flick delivery waits slightly longer before sending input, which gives the target window more time to settle.
- Random key hold timing is a little less machine-perfect.

## Install

Download `OMNAFK-Setup.exe`, run it, and leave **Start with Windows** enabled if you want OMNAFK to live in the tray.
