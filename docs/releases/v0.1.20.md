# OMNAFK v0.1.20

This release makes keepalive behavior easier to see and safer to tune for individual games.

## Changes

- Added a **Gentle only** option to per-game profiles. When enabled, that game receives pointer-based keepalives instead of movement or bound-key input.
- Updated the Roblox preset to use a mouse wiggle so the default keepalive does not move a character or cancel an emote.
- Sightline now previews the next action planned for each marked game.
- Play-aware timing now moves the next send forward while genuine keyboard, mouse, or controller input continues.
- Added probable AFK-kick notices and action-level kick counts to help spot actions a game may be ignoring.
- Moved occasional target commands into an overflow menu and placed advanced profile controls behind **More**.
- Replaced the dormant update blink with a steady attention eye.
- Simplified Keepalive wording and removed the extra caption above Timing.

## Install

Download `OMNAFK-Setup.exe` from this release and run it. Existing settings and target profiles are kept.
