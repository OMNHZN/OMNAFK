## v0.1.4

This release adds monitor placement for marked games and tightens the play-aware hold behavior.

### Added

- Move marked games to a preferred monitor from the General tab.
- Choose placement style: preserve size, maximize, fill work area, or fill monitor.
- Override monitor placement per target from Sightline details.
- Show monitor target and placement status in target details.

### Improved

- Keepalive sends now hold when recent user input is detected, even if Windows reports a different foreground window than expected.
- Monitor moves can wait while the game is active, avoiding surprise repositioning during play.
- GitHub Releases now show cleaner release notes for each update.
