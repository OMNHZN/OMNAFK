OMNAFK

Awake when you aren't.

A quiet Windows tray app that detects game windows, keeps them awake when needed, and stays out of the way when you are playing.

What It Does

Keepalive control without another dashboard to babysit.

OMNAFK watches from the tray. When a game-like window appears, Sightline marks it, explains the detection, and schedules quiet keepalive input. When there is nothing to do, it goes dormant.

Detects

Finds fullscreen, borderless, platform-launched, and known game windows automatically.

Holds

Pauses ticks when recent user input is detected, so it does not fight active play.

Adapts

Learns a small safe key profile per game and shows when adaptive actions are ready.

Product Preview

Sightline shows what OMNAFK sees.

Detection reasons

Each target can show why it was marked, such as known process, game folder, fullscreen, or score threshold.

Target preferences

Mark, ignore, pause, reset learning, or give one target its own interval, action, and monitor rule.

Tray status

The tray menu keeps the important state close: active target count, ignored count, and next tick timing.

User Guide

Start simple. Tune only what you need.

Most users can install OMNAFK, leave the defaults alone, and only open Sightline when something needs a manual mark or ignore.

01
Install and leave it in the tray

Run OMNAFK-Setup.exe. Keep Start with Windows enabled if you want it ready after login.

02
Open Sightline when a game is running

Marked targets receive keepalives. Ignored windows are remembered and stay out of the active list.

03
Use details when something looks wrong

Target details show detection reasons, adaptive learning progress, monitor status, and recent action state.

04
Update from Settings

Settings checks stable GitHub Releases and can open the latest installer when an update is available.

If a game is missed

Open Sightline, switch to All Seen, then mark the window as a game. OMNAFK remembers that target.

If a normal app is marked

Mark it ignored once. The target stays ignored unless you reset overrides.

If adaptive action says learning

Keep playing normally with the game focused. OMNAFK waits for enough safe samples before using that profile.

If input does not land

Check whether the game is running as administrator. OMNAFK may need to run at the same level.

Install

Custom setup. Stable updates.

Download OMNAFK-Setup.exe, run setup, and let the app live quietly in the Windows tray.

Future stable builds are published through GitHub Releases. The app can check for updates and open a structured bug report from Settings.
