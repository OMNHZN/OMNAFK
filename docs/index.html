OMNAFK

Awake when you aren't.

A tray-first Windows utility that automatically detects games and sends quiet keepalive input while staying out of your way.

Download Setup View Source

BEHAVIOR

Install it once. Let it watch.

AUTOMATIC

No start button. OMNAFK detects game-like windows, arms itself when needed, and goes dormant when nothing is running.

QUIET INPUT

Keepalives use background window messages by default, so the app avoids stealing focus from what you are doing.

OVERRIDABLE

Every detected target can be corrected as GAME or IGNORED, and those choices persist across restarts.

SETUP

A single custom installer.

Download OMNAFK-Setup.exe, choose whether it starts with Windows, and finish into the tray.

Future builds are published through GitHub Releases so the Settings tab can check for updates from OMNHZN/OMNAFK.
