# Security Policy

## Supported Version

Security fixes are handled for the latest stable release.

## Reporting A Vulnerability

Please report installer, update, privilege, or execution-safety issues privately through GitHub Security Advisories:

https://github.com/OMNHZN/OMNAFK/security/advisories/new

Do not post exploit details, tokens, or private system information in public issues. Public follow-up can happen after a fix is available.

## Scope

Security-sensitive areas include:

- Installer behavior
- Update download and launch flow
- Elevated restart flow
- Startup registration
- File writes under the OMNAFK config directory
- Community manifest fetch and validation
